Quick help notes

EUMIS Service Chain Editor is currently undergoing user-interface changes.

You may notice several "glitches" with the main area, especially while zoomed out.
Please ignore these issues for now, as they are being worked on.

If your browser freezes for extended periods of time without any noticable action of content being worked out
(For example, when loading a new WSDL file, there is a progress bar, while may lag - but doesn't cause things to freeze)
Then please e-mail the PML Computing Scientist Web Based Visualisation with any specific steps you followed, current WSDL file loaded, and other details that might help track and fix the noticed problem.
Please also state if you can reproduce (repeat) the issue.

Thank you.
